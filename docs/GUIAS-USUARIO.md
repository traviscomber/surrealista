## Cotizador - Tutorial Completo {#cotizador}

### Paso 1: Acceder al Cotizador

1. Desde la página principal, ve a **Herramientas**
2. Selecciona **Cotizador**
3. Verás el formulario de valuación

### Paso 2: Seleccionar Tipo de Propiedad

```
Opciones disponibles:
□ Terreno - Lotes sin construcción
□ Casa - Vivienda unifamiliar
□ Departamento - Propiedad de múltiples unidades
□ Comercial - Locales y oficinas
□ Agrícola - Fundos y propiedades agrícolas
□ Industrial - Propiedades industriales
```

**Consejo**: Si tienes duda, elige la más cercana a tu propiedad.

### Paso 3: Seleccionar Región

```
Regiones disponibles en Chile:
- Arica y Parinacota
- Tarapacá
- Antofagasta
- Atacama
- Coquimbo
- Valparaíso
- Metropolitana (Santiago)
- O'Higgins
- Maule
- Ñuble
- Biobío
- Araucanía
- Los Ríos
- Los Lagos
- Aysén
- Magallanes
```

**Consejo**: Tu región se mapea automáticamente con tus ciudades.

### Paso 4: Ingresar Ciudad (Opcional)

Ingresa la ciudad específica para mayor precisión. Ejemplos:
- Santiago (Metropolitana)
- Valdivia (Los Ríos)
- Valparaíso (Valparaíso)
- Concepción (Biobío)

**Consejo**: Dejar en blanco usa toda la región como referencia.

### Paso 5: Especificar Área en m²

Ingresa el área total de tu propiedad.

```
Ejemplos:
- Terreno pequeño: 500 m²
- Terreno mediano: 5,000 m²
- Terreno grande: 50,000+ m²
- Casa unifamiliar: 150-300 m²
- Departamento: 50-150 m²
```

**Consejo**: Incluye área total construida + patios/terrazas comunes.

### Paso 6: Seleccionar Estado de la Propiedad

```
Opciones:
□ Excelente - Nueva o recién remodelada (+15%)
□ Bueno - Buen estado, sin reparaciones (base)
□ Regular - Requiere algunos arreglos (-12%)
□ Reparación - Necesita inversión (-25%)
□ En construcción - Obra en desarrollo (-15%)
□ Sin mejoras - Terreno sin construcción (-50%)
```

**Consejo**: Sé honesto con el estado para mejor accuracy.

### Paso 7: Agregar Características (Opcional)

Ingresa características separadas por comas:

```
Ejemplos válidos:
- piscina, jardín, estacionamiento
- acceso metro, vista al mar, seguridad
- terraza, bodega, parrilla

Características reconocidas:
piscina, sauna, gimnasio, estacionamiento,
parking, jardín, patio, terraza, vista al mar,
vista privilegiada, acceso metro, transporte
público, seguridad, vigilancia, portería,
conserje, bodega, parrilla
```

**Consejo**: Cada característica añade valor (+1-8%).

### Paso 8: Información Adicional (Opcional)

Ingresa cualquier información relevante:
```
Ejemplos:
- "Ubicación estratégica, cerca de colegios"
- "Vista panorámica, buena iluminación"
- "Acceso por camino privado"
```

### Paso 9: Obtener Valuación

Haz click en el botón verde **"Obtener Valuación"**

El sistema:
1. Analiza tus datos
2. Consulta múltiples fuentes
3. Calcula la valuación
4. Muestra resultados

## Asistente IA - Tutorial Completo {#asistente}

### Accediendo al Asistente

1. Ve a **Herramientas**
2. Selecciona **Asistente IA**
3. Verás una interfaz de chat

### Tipos de Preguntas

**Búsqueda Geográfica**
```
"¿Qué archivos KMZ tengo en Valdivia?"
→ Asistente busca datos de esa región

"Muestra propiedades en Los Lagos"
→ Retorna información de ubicaciones
```

**Análisis de Mercado**
```
"¿Cómo está el mercado en Metropolitana?"
→ Tendencias, precios, análisis regional

"¿Cuáles son las tendencias de precios?"
→ Análisis de mercado y predicciones
```

**Asesoramiento**
```
"¿Dónde debo invertir?"
→ Recomendaciones basadas en datos

"¿Cuáles son las mejores oportunidades?"
→ Análisis de zonas con potencial
```

### Cómo Formular Preguntas

**Lo Mejor**:
```
✓ "¿Cuánto cuesta un terreno en Valdivia?"
✓ "Analiza el mercado en Los Lagos"
✓ "¿Qué zonas tienen potencial de inversión?"
```

**Evitar**:
```
✗ "hola" (muy vago)
✗ "propiedad" (sin contexto)
✗ "cuál es la mejor" (necesita más especificidad)
```

### Interpretar Respuestas

El asistente proporciona:
1. **Respuesta principal** - Información solicitada
2. **Datos verificables** - Fuentes de datos reales
3. **Factores de mercado** - Contexto del análisis

## Casos de Uso {#casos}

### Caso 1: Valuación para Venta

**Objetivo**: Determinar precio de venta de tu propiedad

**Pasos**:
1. Ve al Cotizador
2. Completa datos exactos de tu propiedad
3. Revisa el rango de precio
4. Añade 5-10% si está en excelente estado
5. Usa como precio inicial en anuncio

**Tips**:
- Compara con propiedades similares en el mercado
- Considera la ubicación estratégica
- Resalta características premium

### Caso 2: Análisis de Inversión

**Objetivo**: Evaluar oportunidad de inversión

**Pasos**:
1. Cotiza varios terrenos/propiedades
2. Compara precios/m² entre zonas
3. Usa el Asistente para tendencias
4. Calcula potencial de revalorización
5. Toma decisión basada en datos

**Tips**:
- Busca zonas con tendencia alcista
- Compara con desarrollos planificados
- Considera crecimiento demográfico

### Caso 3: Comparación de Propiedades

**Objetivo**: Elegir entre varias propiedades

**Pasos**:
1. Cotiza cada propiedad por separado
2. Compara precios totales
3. Compara precios/m²
4. Revisa características de cada una
5. Analiza ubicación y mercado

## Troubleshooting {#troubleshooting}

### Problema: "No hay datos disponibles para esta región"

**Causa**: La región tiene pocos registros

**Solución**:
1. Intenta con región más amplia
2. Usa datos de región vecina
3. Consulta al Asistente IA
4. Confía en benchmarks

### Problema: Valuación muy baja o muy alta

**Causa**: Características especiales o datos incompletos

**Solución**:
1. Revisa datos ingresados
2. Verifica estado de propiedad
3. Compara con similares en mercado
4. Solicita tasación oficial

### Problema: El Asistente no entiende mi pregunta

**Causa**: Pregunta muy vaga o mal formulada

**Solución**:
1. Sé más específico
2. Menciona región claramente
3. Especifica tipo de propiedad
4. Pregunta una cosa a la vez

**Ejemplo mejorado**:
```
❌ "¿Cómo está el mercado?"
✓ "¿Cómo está el mercado de terrenos en Biobío?"
```

### Problema: Diferencia grande entre valuación y mercado

**Causa**: Mercado fluctúa o datos desactualizados

**Solución**:
1. Revisa fecha de datos
2. Compara con ofertas actuales
3. Considera características únicas
4. Solicita segunda opinión
