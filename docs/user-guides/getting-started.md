# Getting Started with Sur-Realista

Welcome to Sur-Realista! This guide will help you get started with the platform.

## What is Sur-Realista?

Sur-Realista is a comprehensive real estate management platform that combines property listings, AI-powered analysis, and government data integration to provide a complete solution for real estate professionals and property seekers.

## Key Features

- **Property Management**: Create, edit, and manage property listings
- **AI Assistant**: Get intelligent property analysis and recommendations
- **Government Integration**: Access SII and CIREN data for property verification
- **Multi-role System**: Different access levels for admins, agents, and users
- **Document Management**: Integrate with Google Drive for property documents

## User Roles

### Public Users
- Browse property listings
- Search and filter properties
- View property details
- Contact agents

### Registered Users
- Save favorite properties
- Create property alerts
- Access detailed property information
- Contact agents directly

### Agents
- Manage their property listings
- Access AI analysis tools
- View client inquiries
- Generate property reports

### Administrators
- Manage all properties and users
- Access system analytics
- Configure integrations
- Manage platform settings

## Getting Started

### 1. Account Creation

1. Visit the Sur-Realista website
2. Click "Sign Up" in the top navigation
3. Choose your account type (User/Agent)
4. Fill in your information
5. Verify your email address

### 2. Profile Setup

After creating your account:

1. Complete your profile information
2. Upload a profile picture
3. Set your preferences
4. Configure notification settings

### 3. Exploring Properties

#### Browsing Properties
- Use the main search to find properties
- Apply filters for price, location, type, etc.
- View properties on the map
- Save interesting properties to favorites

#### Property Details
Each property listing includes:
- High-quality photos
- Detailed description
- Price and financial information
- Location and neighborhood data
- Agent contact information
- AI-generated insights (for registered users)

### 4. Using AI Features

Sur-Realista includes powerful AI tools:

#### Property Analysis
- Market value estimation
- Investment potential assessment
- Neighborhood analysis
- Price trend predictions

#### Smart Recommendations
- Properties matching your criteria
- Similar properties in the area
- Investment opportunities
- Market insights

### 5. Contacting Agents

To inquire about a property:
1. Click "Contact Agent" on the property page
2. Fill in your inquiry details
3. Submit your message
4. The agent will respond directly

## For Agents

### Managing Your Listings

1. **Add New Property**
   - Click "Add Property" in your dashboard
   - Fill in all property details
   - Upload high-quality photos
   - Set pricing and availability

2. **Edit Existing Properties**
   - Access your property dashboard
   - Select the property to edit
   - Update information as needed
   - Save changes

3. **AI-Powered Tools**
   - Use AI to generate property descriptions
   - Get market value estimates
   - Analyze investment potential
   - Generate client reports

### Client Management

- View and respond to inquiries
- Track property views and interest
- Generate property reports
- Manage client communications

## Mobile Experience
